Bouncy Ball

Como Jogar

Movimento Básico
  • W: Mover para Frente
  • S: Mover para Trás
  • A: Mover para Esquerda
  • D: Mover para Direita
  • Barra de Espaço: Saltar
  • Mouse: Olhar ao redor
Objetivo

Navegue pelas plataformas e alcance o checkpoint final (amarelo) para completar o nível. Checkpoints vermelhos salvam seu progresso.

Controles do Jogo
  • ESC: Pausar o jogo / Voltar ao menu anterior
  • 1: Mostrar/Esconder UI (Temporizador, FPS, etc.)
  • 2: Ativar/Desativar Noclip (Modo Fantasma)
  • 3: Mostrar/Esconder Checkpoints
